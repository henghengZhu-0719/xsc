# 每日饮食推荐

一个帮助你管理家中食材、记录三餐并智能推荐每日饮食的小工具。

## 项目简介

很多时候我们不知道家里有什么菜、又该怎么搭配三餐才健康均衡。本项目通过维护一个"虚拟冰箱"，记录你拥有的食材，并结合历史饮食记录，为你推荐当天的早、中、晚餐该吃什么。

## 核心功能

### 1. 自定义冰箱

- 添加/删除/编辑你拥有的食材，如鸡蛋、牛肉、猪肉、各类蔬菜等
- 记录食材数量、保质期等信息，便于提醒及时食用
- 按食材分类管理（肉类、蛋类、蔬菜、主食等）

### 2. 三餐记录

- 记录每日早餐、午餐、晚餐的具体饮食内容
- 支持查看历史饮食记录，回顾近期饮食结构

### 3. 智能饮食推荐

- 结合冰箱中现有食材，推荐当天可以做的菜
- 参考之前及当天已吃的饮食，避免重复、保证营养均衡
- 根据早中晚餐的不同特点给出对应建议

## 使用流程

1. 在"我的冰箱"中添加当前拥有的食材
2. 每餐吃完后记录实际饮食内容
3. 下一餐前查看系统给出的推荐，参考冰箱库存进行选择
4. 系统持续根据记录优化推荐结果
